## 📦 Installation & Setup

### Prerequisites
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Code editor (VS Code recommended)
- Live Server extension (recommended)

### Quick Start

**Option 1: Clone the Repository**

    git clone https://github.com/JAY-glit/jaydurga-portfolio.git
    cd jaydurga-portfolio

**Option 2: Download ZIP**

1. Click the green **Code** button on GitHub
2. Select **Download ZIP**
3. Extract and open `index.html` in your browser

### Run with Live Server

**Using VS Code:**
1. Install the **Live Server** extension
2. Right-click `index.html` → **Open with Live Server**
3. Opens at `http://127.0.0.1:5500`

**Using Python:**

    python -m http.server 8000

Then visit `http://localhost:8000`

**Using Node.js:**

    npx serve

## 🔧 Configuration

### 1. Setup Web3Forms (For Contact Form)

1. Visit [web3forms.com](https://web3forms.com)
2. Get your free access key
3. Open `index.html`
4. Find this line and replace with your key:

    <input type="hidden" name="access_key" value="YOUR_ACCESS_KEY_HERE">

### 2. Add Your Resume

1. Save your resume as PDF
2. Name it: `Alladi_Jaydurga_Resume.pdf`
3. Place in `assets/` folder

### 3. Add Your Photos

Replace files in `assets/` folder:
- `profile.jpg` - Casual photo for hero section
- `profile-suit.jpg` - Professional photo for about section

### 4. Customize Personal Info

Edit `index.html`:
- Update name in `<span class="name-3d">` tags
- Update bio in `hero-description`
- Update email in `mailto:` links
- Update phone in `tel:` links
- Update LinkedIn URL

### 5. Customize Skills

In the Skills section of `index.html`:
- Edit skill names and percentages (`data-w="95"`)
- Update star ratings
- Modify descriptions and tags

### 6. Change Theme Colors

Edit `css/styles.css` at the top:

    :root {
        --primary: #6c63ff;
        --secondary: #f50057;
        --accent-1: #00bcd4;
    }
